Custom plugins We provide a Puppet class, ocf::munin::plugin , which installs a custom Munin plugin to a machine, for example, to monitor the number of players on our CS:GO server. Writing a plugin is very easy, should you need to do so.

When called without arguments, it should print to standard output a list of variable names and values: field1.value <value> field2.value <value> ...

When given the lone argument config , it should print display information for Munin graphs and variable warning levels: graph_title Title graph_vlabel yaxis graph_scale no field1.label label field1.warning min:max ...
