The template variables will be automatically filled out by entering text into the input boxes above the main text box. Variables can be located in the template by looking for {<variable name>} before filling it out.

Adding templates New templates can be added to the website by submitting a pull request to the ocf/templates repo. Template files are found in templates/www/templates . Variables can be added in a template as {<variable name>} .
