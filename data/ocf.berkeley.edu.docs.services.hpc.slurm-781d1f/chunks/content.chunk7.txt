Your job's ID is output when a batch script is submitted, or you can find it using squeue (more details below). Viewing Slurm info To view the queue of running and pending jobs from all users, run squeue . To see the details of one job, run squeue -j [job ID] . To view the list of all HPC nodes, and some details about them, run sinfo -N -l .
