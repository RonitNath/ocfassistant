Virtual hosting mail allows groups to receive mail at @group.b.e addresses, and send from those same addresses. It complements our web hosting nicely. Features Each user has mail forwarding. Each virtual user can have its own password (not the same as the account password). If a user doesn't have a password set, then they can receive mail but not send it. Simple admin panel built into ocfweb for admins to set passwords for emails.
