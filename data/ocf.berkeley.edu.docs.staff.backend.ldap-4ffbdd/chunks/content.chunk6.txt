To add an account, first create a file (we call it user_file ): dn: uid=asdf,ou=People,dc=OCF,dc=Berkeley,dc=EDU objectClass: ocfAccount objectClass: account objectClass: posixAccount cn: asdf uid: asdf uidNumber: 25444 gidNumber: 20 homeDirectory: /home/a/as/asdf loginShell: /bin/bash calnetUid: 758472 Then authenticate with Kerberos : $ kinit myusername/admin Finally run ldapadd : $ ldapadd < user_file This also works on lists of entries to add separated by empty newlines.
